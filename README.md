# Suchi - 数値統計ツール

## 概要
`Suchi` は、標準入力またはファイルから数値データを受け取り、以下の統計情報を計算して出力するシンプルなコマンドラインツールです：

- 平均
- 中央値
- 標準偏差
- 分散

## 動作環境・使用ソフトウェア
- このツールは Python で実装されており、以下の環境で動作確認済み：
  - Ubuntu 22.04
  - Python 3.8.10 以上
